Pythonの基本構造

式と文

Pythonでは、値を生成するものを「式」処理を実行するものを「文」と呼ぶ。
例（式と文）
    < 式 >
  • 24
  • 3 + 7
  • "apple"
  • [2, 3, 5, 7]
  • len(list)
  • score <= 10
    •
    < 文 >
  • score = 24
  • return 3 + 7
  • print("apple")
  • list.append(11)
  • if score <= 10:
        pass
    •

コメント

「#」から行末まではコメントとみなされ、Pythonの文とは解釈されずに無視される。
プログラム内に説明や注釈を追加するためのメモのようなもの。 
		
# 素数判定関数
def is_prime(i):
    if i <= 1:                            # 1以下ならば素数でない
        return False
    for j in range(2, int(i**0.5) + 1):   # 順に割って割り切れたら素数でない
        if i % j == 0:
            return False
    return True                           # そうでなければ素数

出力と入力

標準出力と標準入力

最も一般的である、ディスプレイへの出力を標準出力、キーボードからの入力を標準入力という。 Pythonでは、標準出力のための print関数 、標準入力のための input関数 、が用意されている。

print 関数

			
def print(
    *objects: object,
    sep: str = " ",
    end: str = "\n"
) -> None
引数 型 初期値 説明
*objects object 表示したいオブジェクト
sep str " " 区切り文字
end str "\n" 末尾に付ける文字
コンマ区切りで指定された *objects の区切り文字を sep にし、末尾に end を付け加えた文字列を表示する。
※「*」は可変長引数を意味する記号である。任意の数の引数をコンマ区切りで渡すことができる。
※「\n」は改行を意味するエスケープシーケンスである。環境によってはバックスラッシュ「\」が日本円記号「¥」と表示される場合がある。 
			
print("apple")                              # apple
print("app", "le")                          # app le
print("app", "le", sep = "-")               # app-le
print("b", "n", "na", sep = "a")            # banana
print("b", "n", "n", sep = "a", end = "a")  # banana
print("a", "b", "c", sep = "-", end = "!")  # a-b-c!

input 関数

			
def input(
    prompt: object = ""
) -> str
prompt を表示した後に、キーボードからの入力を受け取り、それを返す。 
			
print(input())                    # ユーザーの入力を表示
print(input("あなたの名前は？"))      # ユーザーの名前を聞いてから入力させてそれを表示

リテラル

プログラム内で直接記述される値をリテラルという。 以降、初歩的なリテラルを紹介していく。

数値リテラル

Pythonでは、数値型は全部で３つあり、整数(int)、浮動小数点数(float)、複素数(complex)である。 「int」は「integral」の略である。 複素数リテラルを表す場合、虚数単位は「j」を用いる。また、数値リテラルに含まれるアンダースコア「_」は無視される。
例（数値リテラル）
    < int >
  • 24
  • -3735
  • 0
  • 134_468
    < float >
  • 23.364
  • 0.0
  • 0.348_35
  • -5.0
    < complex >
  • 4j
  • 3-7j
  • 0j
  • 43+31_576j

文字列リテラル

文字列型(str) は「string」の略である。文字列リテラルを表す場合、表示したい文字をシングルクォーテーション「'」またはダブルクォーテーション「"」で囲む。 「""」または「''」を空文字列という。
例（文字列リテラル）
  • "apple"
  • ""
  • '円周率'
  • "山田"
  • "3872"
  • '1+1'

真偽値リテラル

真偽値型(bool) は「True」または「False」のどちらかの値のみを持つ。 「bool」は「boolean」の略である。 「True」は「真」を意味し、「False」は「偽」を意味する。 「True」と「False」はそれぞれ真偽値型のリテラルである。

NoneTypeリテラル

NoneType型は「None」のみの値を持つ。 「None」は存在しないことを表す。 「None」はNoneType型のリテラルである。

type 関数

Pythonでは、オブジェクトの型を調べるための関数として type関数 が用意されている。 引数に渡したオブジェクトの型を返す。 
			
print(type(5))          # <class 'int'>
print(type("i"))        # <class 'str'>
print(type(8.26))       # <class 'float'>
print(type(True))       # <class 'bool'>

整数の生成

intコンストラクタの引数にfloat型の値を入れると、その値を切り捨ててできた整数が生成できる。
※コンストラクタとは、型（クラス）の名前と等しい名前の関数のことである。この関数は特殊な意味を持つ。 
			
print(int(32.65))            # 32
print(int(-3.54))            # -3
print(int(6.993))            # 6
intコンストラクタの引数に整数を表すstr型の値を入れると、その整数が生成できる。 
			
print(int("32"))             # 32
print(int("-3"))             # -3
print(int("628"))            # 628

小数の生成

floatコンストラクタの引数にint型の値を入れると、その値をfloat型に変換したものが生成できる。 
			
print(float(32))             # 32.0
print(float(-5))             # -5.0
print(float(280))            # 280.0
floatコンストラクタの引数に小数を表すstr型の値を入れると、その小数が生成できる。 
			
print(float("63.29"))          # 63.29
print(float("-46.3"))          # -46.3
print(float("94.233"))         # 94.233

複素数の生成

complexコンストラクタを用いて、第一引数に実部、第二引数に虚部を入れることで複素数を生成できる。 
			
def complex(
    real: number = 0
    imag: number = 0
) -> complex
		
			
print(complex(-2, 4))          # -2+4j
print(complex(7.3, 8.4))       # 7.3+8.4j
print(complex(3))              # 3+0j
print(complex(-19.5))          # -19.5+0j
print(complex(0, -2))          # -2j
print(complex(0, 45.6))        # 45.6j
complexコンストラクタの引数に複素数を表すstr型の値を入れると、その複素数が生成できる。 
			
print(complex("84-47j"))          # 84-47j
print(complex("28+3j"))           # 28+3j
print(complex("-5-28j"))          # -5-28j

複素数の実部・虚部の取得

complexの real, imag プロパティを用いて、複素数の実部, 虚部を取得することができる。 
			
print((2-5j).real)               # 2
print((-3+2j).real)              # -3
print((2-5j).imag)               # -5
print((-3+2j).imag)              # 2

数値型における比較演算子

int または float に対して以下の比較演算子を用いると、真ならばTrue、偽ならばFalseが返される。

演算子の優先順位

演算子には以下のような優先順位がある。 表の上が優先順位が高く、下に行くにつれて低くなる。 同じ優先度の演算では、左から右に演算が行われる。 また、カッコを付けることでその部分を優先的に演算できる。
演算子 説明
** べき乗
+x , -x 正数、負数
* , / , // , % 乗算、除算、切り捨て除算、剰余算
+ , - 加算、減算
> , < , >= , <= , == , != 比較 
			
print(2+3*-2**2)                       # -10
print(3**2%6+3/2)                      # 4.5
print(2*-3*-4-3**2)                    # 15
print(2*3-4*5>2-3**2)                  # False
print((2+3)*2**3//6<=4-2*(3-2)+4)      # True
print(-((4-3)*3)**2>=3*7%3*(4-2))      # False

数値に関する関数

abs関数：引数に渡した数値の絶対値を返す。 
			
print(abs(2))        # 2
print(abs(-5.2))     # 5.2
print(abs(3+4j))     # 5.0
round関数：第一引数に渡した数値を第二引数で指定した小数の位で丸め込む。 第二引数が無い場合は整数に丸め込む。
※丸め込みは四捨五入に似たようなものだが、厳密には異なる。 
			
def round(
    number: int | float,
    ndigits: int = None
) -> int | float
		
			
print(round(2.32))        # 2      (整数)
print(round(-1.387, 2))   # -1.39  (小数第二位)
print(round(71.31, 1))    # 71.3   (小数第一位)
print(round(3587, -2))    # 3600   (百の位)

メソッド

それぞれの型に特有の関数のことをメソッドという。
関数とメソッドの使い方の違い
　　関数の使い方： 関数名 ( 対象のオブジェクト )
　　　　　　　例： abs ( 4+3j )
　　　　　　説明： 4+3j の絶対値を取得

メソッドの使い方： 対象のオブジェクト . メソッド名()
　　　　　　　例： 4+3j . conjugate()
　　　　　　説明： 4+3j の共役な複素数を取得
関数：関数に対して対象のオブジェクトを渡す
メソッド：対象のオブジェクトに対してメソッドを適用する
以降、文字列型のメソッドを紹介していく。

文字列の置換

			
def replace(
    old: str,      # 置換したい元の文字列
    new: str       # 置換後の文字列
) -> str

部分文字列の検索

指定した文字列が最初に（最後に）現れる位置を表示。
メソッド 検索条件 無い場合

論理演算子

真偽値の演算を行うときには論理演算子を用いる。 真ならば True、偽ならば False が返される。
表記 意味
x or y 少なくとも一方がTrue
x and y x, y 共にTrue
not x x がFalse 
			
print(True or True)        # True
print(True or False)       # True
print(False or True)       # True
print(False or False)      # False

print(True and True)       # True
print(True and False)      # False
print(False and True)      # False
print(False and False)     # False

print(not True)            # False
print(not False)           # True

演算子の優先順位

先ほどの演算子の優先順位の表に論理演算子を加えると以下のようになる。表の上の方が優先度が高い。
演算子 説明
** べき乗
+x , -x 正数、負数
* , / , // , % 乗算、除算、切り捨て除算、剰余算
+ , - 加算、減算
> , < , >= , <= , == , != 比較
not x 論理演算子 not
and 論理演算子 and
or 論理演算子 or 
			
print(not 3 + 2 >= -3 + 2 * 2 ** 2 and "a" == "a")   # False
print(True or 2 ** 2 == 4 and not True)              # True
print("a" > "2" and 3 * 2 <= 4 ** 2 // 3 and True)   # False
print(not abs(-2) < 1 and 3 * 5 != 4 ** 2 - 1)       # False
print(not False and False or True and not False)     # True
print(5-2 == -abs(2-5) or False or not "The"<"the")  # False

比較演算子の連鎖

比較演算子は任意に連鎖できる。
「x < y < z」は、「x < y and y < z」と同じ意味。
「x < y > z」は、「x < y and y > z」と同じ意味。 
			
print(1 < 2 < 3)           # True
print(3 > 6 > 5)           # False
print(1 <= 2 <= 3)         # True
print(5 <= 8 > 9)          # False
print(1.2 > 0.0 <= 3.5)    # True
print(5.2 >= -3 < -6.2)    # False
print(2.3 == 2.3 < 5.7)    # True
print(2.5 < 0.2 == 0.2)    # False

変数

変数への値の代入

変数とは、値に対して名前を紐づけ、その名前によって値を扱うことができるもの。 「 変数名 = 値 」の形で変数を定義し、値を代入できる。 この「=」を代入演算子という。 
			
a = 15               # aという変数を用意し、それに15を代入
b = 25               # bという変数を用意し、それに25を代入

print(a)             # 15
print(a * 2 - b)     # 5
再び値を代入したときには、値は上書きされる。 
			x = "apple"          # xという変数を用意し、それにappleを代入
print(x)             # apple
x = "banana"         # 変数xにbananaを代入（上書きされる）
print(x)             # banana
右辺が変数であっても問題ない。 
			a = 10               # aという変数を用意し、それに10を代入
b = a                # bという変数を用意し、それにaの値を代入
print(b)             # 10

コレクション

コレクションとは

以下の３つの条件を満たすデータ型をコレクションという。

  1. 複数のオブジェクトを格納できる（in演算子が使える）
  2. 要素を一つずつ取り出せる（for文で繰り返し処理ができる）
  3. 格納してあるオブジェクトの個数を取得できる（len関数を使える）

また、１つ目の条件を満たすデータ型をコンテナ、２つ目の条件を満たすデータ型はイテラブルであるという。

コレクションの分類

コレクション(collection)は主に、シーケンス(sequence)、セット(set)、マッピング(mapping)に分類される。

シーケンス
番号(index)によって要素を取り出すことができる。
セット（集合）
集合のこと。よって要素の順番がなく、重複もない。
マッピング
キー(key)によって値(value)を取り出すことができる。


ミュータブル（要素の変更が可能）かどうかによっても分類できる。
ミュータブル：赤色　　　イミュータブル：青色

リスト

リスト(list)は、ミュータブルなシーケンスである。要素をコンマ区切りで表したものを[ ]で囲ったものがリテラルになる。 末尾に余分なコンマが一つ付いていてもよい。
例（リストのリテラル）
  • [2, 3, 5, 7, 11, 13, 17, 19]
  • [-4.5, -0.8, 2.9, 3.8, 4.2,]
  • ['apple', 'banana', 'orange']
  • [2, -3.7, "text", True, "text", 2,]
  • ['a']
  • []

タプル

タプル(tuple)は、イミュータブルなシーケンスである。要素をコンマ区切りで表したものを( )で囲ったものがリテラルになる。 タプルの場合は( )で囲まなくても問題はないが、見やすくなるので通常は囲む。末尾に余分なコンマが一つ付いていてもよい。 単要素のタプルを表す場合には、末尾にコンマが必要。
例（タプルのリテラル）
  • 2, 3, 5, 7, 11, 13, 17, 19
  • (-4.5, -0.8, 2.9, 3.8, 4.2,)
  • 'apple', 'banana', 'orange',
  • (2, -3.7, "text", True, "text", 2)
  • ('a',)
  • ()

文字列

文字列型(str)は、テキストシーケンスとも呼ばれる、イミュータブルなシーケンスである。 例えば、 "apple" は、"a" "p" "p" "l" "e" という５つの要素を格納したものである。 文字列リテラルについては、前に説明したので省く。

range

rangeは、等差型の整数のみを取り扱うイミュータブルなシーケンスである。 リテラルは無いので、後述するrangeコンストラクタを用いることでしか生成できない。 rangeオブジェクトは全ての値を格納しているのではなく、必要に応じてその都度計算して返す。 なので他のコレクションに比べて非常にメモリ消費が少なく、要素の検索が高速である。
例（rangeをリストに変換したもの）
  • [0, 1, 2, 3, 4, 5, 6, 7, 8]
  • [0, 2, 4, 6, 8, 10, 12]
  • [-11, -7, -3, 1, 5, 9, 13]
  • [25, 20, 15, 10, 5, 0, -5]
  • [2]
  • []
※　上に見せているものはあくまでrangeをリストに変換したものなのでリストである。rangeではない。

set

setは、ミュータブルな集合である。要素をコンマ区切りで表したものを{ }で囲ったものがリテラルになる。 末尾に余分なコンマが一つ付いていてもよい。空のsetを表すリテラルはない。後述するsetコンストラクタを用いて、空のsetを生成できる。
例（setのリテラル）
  • {2, 3, 5, 7, 11, 13, 17, 19,}
  • {-4.5, -0.8, 2.9, 3.8, 4.2}
  • {'apple', 'banana', 'orange',}
  • {2, -3.7, "text", True, -8j}
  • {'a'}

frozenset

frozensetは、イミュータブルな集合である。リテラルは無いので、後述するfrozensetコンストラクタを用いることでしか生成できない。

辞書

辞書(dict)は、ミュータブルなマッピングである。「dict」は「dictionary」の略である。「key: value」の組をコンマ区切りで表したものを{ }で囲ったものがリテラルになる。 末尾に余分なコンマが一つ付いていてもよい。
例（辞書のリテラル）
  • {"apple": 180, "peach": 100, "kiwi": 80}
  • {"山田": 90, "小林": 30, "佐藤": 70,}
  • {"India": 14_4170, "China": 14_2520,}
  • {"diabetes": "糖尿病", "pneumonia": "肺炎"}
  • {}

所属検査演算子

所属検査演算子を用いて、ある要素がコンテナに含まれているかどうかを調べることができる。所属検査演算子は比較演算子の一種である。
表記 意味
a in container a は container に含まれる
a not in container a は container に含まれない 
			
print(7 in [2, 3, 5, 7, 11, 13, 17])            # True
print(6 in (2, 3, 5, 7, 11, 13, 17))            # False
print(7 not in {2, 3, 5, 7, 11, 13, 17})        # False
print(6 not in [2, 3, 5, 7, 11, 13, 17])        # True
str の場合は部分文字列の検索ができる 
			print("n" in "encounter")                       # True
print("count" in "encounter")                   # True
print("enter" not in "enterprise")              # False
print("lemon" not in "enterprise")              # True
str 以外の場合は部分的な検索はできない 
			print([3, 5] in [2, 3, 5, 7, 11, 13])           # False
print({7, 11} in {2, 3, 5, 7, 11, 13})          # False
dict の場合は key の検索となる 
			print("apple" in {"apple": 120, "lemon": 70})   # True
print("banana" in {"apple": 120, "lemon": 70})  # False
dict の value の検索はできない 
			print(120 in {"apple": 120, "lemon": 70})       # False
print(70 in {"apple": 120, "lemon": 70})        # False

イテラブルオブジェクトに関する関数

Pythonには、引数にイテラブルオブジェクトを取る関数がいくつか用意されている。 以降、それらを紹介していく。

sum関数

引数に渡されたイテラブルオブジェクトの要素の合計を返す。 
			
def sum(
    iterable: iterable,
    start: number = 0
) -> number
		
			
print(sum([2, 3, 5, 7, 11, 13]))         # 41
print(sum((2, 3, 5, 7, 11, 13)))         # 41
print(sum({2, 3, 5, 7, 11, 13}))         # 41
dict の場合は key の合計が返される。 
			print(sum({8:'d9m6', 12:'b3w9', 15:'m2yq'}))  # 35
print(sum({-3: 23, 0: 46, 5: 58, 8: 65}))     # 10
初期値(start)を設定することで、初期値に各要素を足し合わせたものが返される。 
			print(sum([2, 3, 5, 7, 11, 13], start = -5))  # 36
print(sum((2, 3, 5, 7, 11, 13), start = 9))   # 50
print(sum({2, 3, 5, 7, 11, 13}, -16))         # 25
print(sum({7:'e7tm', 13:'u4kg'}, 10))         # 30
数値でないものが含まれている場合、TypeErrorが返される 
			print(sum([4, 8, -3, 'a', 6, '3', 'x']))      # TypeError
print(sum({'AA':28, 'BB':36, 'CC':48}))       # TypeError

max, min関数

max関数
引数に渡されたイテラブルオブジェクトの要素の最大値を返す。
min関数
引数に渡されたイテラブルオブジェクトの要素の最小値を返す。
			
print(max([2, 3, 5, 7, 11, 13]))           # 13
print(min((2, 3, 5, 7, 11, 13)))           # 2
print(max({'2', '7', 'C', 'e', 'h'}))      # h
print(min(['2', '7', 'C', 'e', 'h']))      # 2
print(max("Tg28a47f"))                     # g
print(min("Tg28a47f"))                     # 2
dict の場合は key の最大値・最小値が返される。 
			print(max({-4: 212, 5: 251, 12: 263}))     # 12
print(min({-4: 212, 5: 251, 12: 263}))     # -4
引数のイテラブルオブジェクトが空の場合は、default の値が返される。 default が設定されていないときには ValueError が返される。 
			print(min((), default = "empty"))          # empty
print(max([]))                             # ValueError
print(max({3, 7}, default = "empty"))      # 7
異なる型の要素が含まれている場合、TypeErrorが返される。 
			print(max({-2, 8, 4, '4', 'a', -3}))       # TypeError
print(min({-10: '1', "y": '2', "x": '3'})) # TypeError

all, any関数

引数に渡したイテラブルオブジェクトについて、次の条件を満たせばTrue、でなければFalseが返される。
関数 条件 空の場合
all 全ての要素がTrue Trueが返される
any いずれかの要素がTrue Falseが返される
※　「要素がTrue」＝「boolコンストラクタを用いて真偽値型に変換したときにTrueになる」
※　「空の場合」＝「引数に渡したイテラブルオブジェクトが空の場合」 
			
print(all([0, 2, 4, 6, 8, 10]))            # False
print(any({0, 2, 4, 6, 8, 10}))            # True
print(all({0, 0.0, 0j, "", None}))         # False
print(any((0, 0.0, 0j, "", None)))         # False
print(all("a b c"))                        # True
print(any("apple"))                        # True
dict の場合は key によって判断される。 
			print(all({5: '', 7: False, 15: 0.0}))     # True
print(any({0: None, 8: "", 12: 0j}))       # True
引数のイテラブルオブジェクトが空の場合はこのようになる。 
			print(all([]))                             # True
print(any({}))                             # False

sorted関数

引数に渡したイテラブルオブジェクトを並び変えた、新たなリストを返す。 
			
def sum(
    iterable: iterable,

reverse を指定しないと、昇順に並び替えができる。 
			
print(sorted([4, 6, -7, 3, 8, -5, -7, 6]))
# [-7, -7, -5, 3, 4, 6, 6, 8]
reverse = False で昇順、reverse = True で降順に並び替えできる。 
			print(sorted((4, 6, -7, 3, 8, -5, -7, 6), reverse = True))
# [8, 6, 6, 4, 3, -5, -7, -7]

print(sorted({'e', '7', 'M', 'H', 'G'}, reverse = False))
# ['7', 'G', 'H', 'M', 'e']

print(sorted("W3x8U4B2", reverse = True))
# ['x', 'W', 'U', 'B', '8', '4', '3', '2']
dict の場合は、key のみが対象となる。 
			print(sorted({4:'a', 5:'b', 7:'c', 3:'d', 9:'e'}))
# [3, 4, 5, 7, 9]
異なる型の要素が含まれている場合、TypeErrorが返される。 
			print(sorted([4, "n", -7, "k", 8, "d", -7, 6]))
# TypeError
