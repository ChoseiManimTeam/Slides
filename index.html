Pythonの基本構造

式と文

Pythonでは、値を生成するものを「式」処理を実行するものを「文」と呼ぶ。
例（式と文）
    < 式 >
  • 24
  • 3 + 7
  • "apple"
  • [2, 3, 5, 7]
  • len(list)
  • score <= 10
    •
    < 文 >
  • score = 24
  • return 3 + 7
  • print("apple")
  • list.append(11)
  • if score <= 10:
        pass
    •

インデント

if・while・for・def 文などにおいて、それぞれのまとまりを作るために行頭に空白を入れて字下げを行う事をインデントという。
インデントは原則半角スペース4つ分で行う。 
			
while True:
    hoge += 1    # この行の初めに、半角スペース４つがある

コメント

「#」から行末まではコメントとみなされ、Pythonの文とは解釈されずに無視される。
プログラム内に説明や注釈を追加するためのメモのようなもの。 
		
# 素数判定関数
def is_prime(i):
    if i <= 1:                            # 1以下ならば素数でない
        return False
    for j in range(2, int(i**0.5) + 1):   # 順に割って割り切れたら素数でない
        if i % j == 0:
            return False
    return True                           # そうでなければ素数

出力と入力

標準出力と標準入力

最も一般的である、画面への出力を標準出力、キーボードからの入力を標準入力という。 Pythonでは、標準出力のための print関数 、標準入力のための input関数 、が用意されている。

print 関数

			
def print(
    *objects: object,
    sep: str = " ",
    end: str = "\n"
) -> None
引数 型 初期値 説明
*objects object 表示したいオブジェクト
sep str " " 区切り文字
end str "\n" 末尾に付ける文字
コンマ区切りで指定された *objects の区切り文字を sep にし、末尾に end を付け加えた文字列を表示する
※「*」は可変長引数を意味する記号である。任意の数の引数をコンマ区切りで渡すことができる。
※「\n」は改行を意味するエスケープシーケンスである。環境によってはバックスラッシュ「\」が日本円記号「¥」と表示される場合がある。 
			
print("apple")                              # apple
print("app", "le")                          # app le
print("app", "le", sep = "-")               # app-le
print("b", "n", "na", sep = "a")            # banana
print("b", "n", "n", sep = "a", end = "a"   # banana
print("a", "b", "c", sep = "-", end = "!")  # a-b-c!

input 関数

			
def input(
    prompt: object = ""
) -> str
prompt を表示した後に、キーボードからの入力を受け取り、それを返す 
			
print(input())                    # ユーザーの入力を表示
print(input("あなたの名前は？"))      # ユーザーの名前を聞いてから入力させてそれを表示

リテラル

プログラム内で直接記述される値をリテラルという。 以降、初歩的なリテラルを紹介していく。

数値リテラル

Pythonでは、数値型は全部で３つあり、整数(int)、浮動小数点数(float)、複素数(complex)である。 「int」は「integral」の略である。 複素数リテラルを表す場合、虚数単位は「j」を用いる。また、数値リテラルに含まれるアンダースコア「_」は無視される。
例（数値リテラル）
    < int >
  • 24
  • -3735
  • 0
  • -13_44_68
    < float >
  • 23.364
  • 0.0
  • -0.34_35
  • -5.0
    < complex >
  • 4j
  • 3-7j
  • 0j
  • 31_56-44_82j

文字列リテラル

文字列型(str) は「string」の略である。文字列リテラルを表す場合、表示したい文字をシングルクォーテーション「'」またはダブルクォーテーション「"」で囲む。
例（文字列リテラル）
  • "apple"
  • 'B'
  • '円周率'
  • "山田"
  • "3872"
  • "1+1"

真偽値リテラル

真偽値型(bool) は「True」または「False」のどちらかの値のみを持つ。 「bool」は「boolean」の略である。 「True」は「真」を意味し、「False」は「偽」を意味する。 「True」と「False」はそれぞれ真偽値型のリテラルである。

NoneTypeリテラル

NoneType型は「None」のみの値を持つ。 「None」は存在しないことを表す。 「None」はNoneType型のリテラルである。

type 関数

Pythonでは、オブジェクトの型を調べるための関数として type関数 が用意されている。 
			
def type(
    object: object
) -> type          # 入力したobjectの型が返される
		
			
print(type(5))          # <class 'int'>
print(type("i"))        # <class 'str'>
print(type(8.26))       # <class 'float'>
print(type(True))       # <class 'bool'>

演算子の優先順位

演算子には以下のような優先順位がある。 表の上が優先順位が高く、下に行くにつれて低くなる。 同じ優先度の演算では、左から右に演算が行われる。 また、カッコを付けることでその部分を優先的に演算できる。
演算子 説明
** べき乗
+x , -x 正数、負数
* , / , // , % 乗算、除算、切り捨て除算、剰余算
+ , - 加算、減算
> , < , >= , <= , == , != 比較 
			
print(2+3*-2**2)                       # -10
print(3**2%6+3/2)                      # 4.5
print(2*-3*-4-3**2)                    # 15
print(2*3-4*5>2-3**2)                  # False
print((2+3)*2**3//6<=4-2*(3-2)+4)      # True
print(-((4-3)*3)**2>=3*7%3*(4-2))      # False

数値に関する関数

abs関数：引数に渡した数値の絶対値を返す。 
			
def abs(
    x: number
) -> number
		
			
print(abs(2))        # 2
print(abs(-5.2))     # 5.2
print(abs(1+1j))     # 1.41421356…
round関数：第一引数に渡した数値を第二引数で指定した小数の位で丸め込む。 第二引数が無い場合は整数に丸め込む。
※丸め込みは四捨五入に似たようなものだが、厳密には異なる。 
			
def round(
    number: number,
    ndigits: int = None
) -> number
		
			
print(round(2.32))        # 2      (整数)
print(round(-1.387, 2))   # -1.39  (小数第二位)
print(round(71.31, 1))    # 71.3   (小数第一位)
print(round(3587, -2))    # 3600   (百の位)

メソッド

それぞれの型に特有の関数のことをメソッドという。
関数とメソッドの使い方の違い
　　関数の使い方： 関数名 ( 対象のオブジェクト )
　　　　　　　例： abs ( 4+3j )
　　　　　　説明： 4+3j の絶対値を取得

メソッドの使い方： 対象のオブジェクト . メソッド名()
　　　　　　　例： 4+3j . conjugate()
　　　　　　説明： 4+3j の共役な複素数を取得
関数：関数に対して対象のオブジェクトを渡す
メソッド：対象のオブジェクトに対してメソッドを適用する
以降、文字列型のメソッドを紹介していく。

文字列の置換

			
def replace(
    old: str,      # 置換したい元の文字列
    new: str       # 置換後の文字列
) -> str
		
			
print("Hello World".replace("World", "Python"))
# Hello Python

print("Yamada Taro".replace("Yamada", "Tanaka"))
# Tanaka Taro

部分文字列の判定

指定した文字列で始まる（終わる）ならばTrue、そうでなければFalseを返す。
メソッド 条件
startswith 指定した文字列で始まる
endswith 指定した文字列で終わる 
			print("Hello World".startwith("Hello"))    # True
print("Hello World".endswith("World"))     # True

部分文字列の検索

指定した文字列が最初に（最後に）現れる位置を表示。
メソッド 検索条件 無い場合
find 最初 -1を返す
rfind 最後 -1を返す
index 最初 ValueErrorを返す
rindex 最後 ValueErrorを返す 
			print("Hello World".find("o"))     # 4
print("Hello World".find("a"))     # -1
print("Hello World".rfind("o"))    # 7
print("Hello World".rfind("a"))    # -1
print("Hello World".index("o"))    # 4
print("Hello World".indes("a"))    # ValueError
print("Hello World".rindex("o"))   # 7
print("Hello World".rindex("a"))   # ValueError

真偽値型

真偽値の生成

コンストラクタを用いて様々なオブジェクトを真偽値型に変換できる。 次のいずれかに該当すればFalse、しなければTrueが返される。

  • 偽であると定義されている定数：False、None
  • 数値型におけるゼロ：0、0.0、0j
  • 空のイテラブルオブジェクト：( )、[ ]、{ }、set( )、range(0)
			
print(bool(24))            # True
print(bool(0.0))           # False
print(bool(None))          # False
print(bool(3.74))          # True

論理演算子

真偽値の演算を行うときには論理演算子を用いる。 真ならば True、偽ならば False が返される。
表記 意味
x or y 少なくとも一方がTrue
x and y x, y 共にTrue
not x x がFalse 
			
print(True or True)        # True
print(True or False)       # True
print(False or True)       # True
print(False or False)      # False

print(True and True)       # True
print(True and False)      # False
print(False and True)      # False
print(False and False)     # False

print(not True)            # False
print(not False)           # True

演算子の優先順位

先ほどの演算子の優先順位の表に論理演算子を加えると以下のようになる。表の上の方が優先度が高い。
演算子 説明
** べき乗
+x , -x 正数、負数
* , / , // , % 乗算、除算、切り捨て除算、剰余算
+ , - 加算、減算
> , < , >= , <= , == , != 比較
not x 論理演算子 not
and 論理演算子 and
or 論理演算子 or 
			
print(not 3 + 2 >= -3 + 2 * 2 ** 2 and "a" == "a")   # False
print(True or 2 ** 2 == 4 and not True)              # True
print("a" > "2" and 3 * 2 <= 4 ** 2 // 3 and True)   # False
print(not abs(-2) < 1 and 3 * 5 != 4 ** 2 - 1)       # False
print(not False and False or True and not False)     # True
print(5-2 == -abs(2-5) or False or not "The"<"the")  # False

比較演算子の連鎖

比較演算子は任意に連鎖できる。
「x < y < z」は、「x < y and y < z」と同じ意味。
「x < y > z」は、「x < y and y > z」と同じ意味。 
			
print(1 < 2 < 3)           # True
print(3 > 6 > 5)           # False
print(1 <= 2 <= 3)         # True
print(5 <= 8 > 9)          # False
print(1.2 > 0.0 <= 3.5)    # True
print(5.2 >= -3 < -6.2)    # False
print(2.3 == 2.3 < 5.7)    # True
print(2.5 < 0.2 == 0.2)    # False
